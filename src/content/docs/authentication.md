---

Для доступа к API нужен персональный токен. Он выдаётся по email и передаётся в заголовке `Token` в каждом запросе.

## Как использовать токен

Передавайте токен в заголовке `Token`:

```bash
curl https://api.documentat.io/api/prod/users \
  -H "Token: <ваш токен>"
```

Запросы без заголовка или с невалидным токеном возвращают `401 Unauthorized`.
