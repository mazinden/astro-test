---
title: Обзор
description: Краткое описание API для работы с пользователями.
---

Users API — это REST-сервис для управления пользователями: создание, чтение, обновление и удаление записей.

Запросы выполняются по адресу `https://api.documentat.io/api/prod` (для разработки доступен `…/api/dev`). Описание всех эндпоинтов и схем данных доступно в разделе API.
